== Architecture Notes ==
- svelte.config.js uses adapter-cloudflare (not adapter-node)
- Supabase clients use lazy initialization (getSupabase/getSupabaseAdmin) to avoid build-time crashes
- wrangler.toml must include pages_build_output_dir and nodejs_compat flag
- Frontend env vars: PUBLIC_SUPABASE_URL, PUBLIC_SUPABASE_ANON_KEY, PRIVATE_SUPABASE_SERVICE_KEY
- Ingest env vars: SUPABASE_URL, SUPABASE_SERVICE_KEY, REDIS_URL
- signals_1min is a materialized view refreshed by worker (not TimescaleDB continuous aggregate)
- Auth uses Supabase Auth with @supabase/ssr for cookie-based sessions on Cloudflare Pages
- Stripe billing: Pro ($19/mo), Team ($49/mo) - products created in Stripe dashboard
- Admin access: is_admin flag on users table, no role-based system
